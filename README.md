# org.w3c.mathml3

Defines a DITA-OT plugin containing the W3C MathML schemas.

## Installation

1. Checkout this repository in ```{DITA_OT_PATH}/plugins/org.w3c.mathml3```
2. From ```{DITA_OT_PATH}``` run ```ant -f integrator.xml```
